## Context
Emberwalk is a lightweight web research tool (CLI + MCP) that replaces Firecrawl for simple research workflows. It searches, fetches pages with anti-bot defaults, extracts clean markdown, and outputs a directory of .md files. Built in a single session on 2026-06-17.

## Key Decisions Already Made
- Name: "emberwalk" (riff on Firecrawl — ember = lighter fire, walk = lighter than crawl)
- Single file, not a package — emberwalk.py does everything
- DuckDuckGo default, Brave Search optional via BRAVE_API_KEY env var
- Tiered fetch: simple HTTP first, Playwright only when content < 200 chars
- Output: directory of .md files + index.md (not a single combined file)
- MCP server uses FastMCP + stdio, same pattern as PIF
- Project lives at ~/Documents/emberwalk/ (not in misc/)

## Pitfalls to Avoid
- The search package is `ddgs`, NOT `duckduckgo-search` — the old name is deprecated and silently returns 0 results
- pip on macOS requires a venv — system pip refuses to install (PEP 668)
- MCP config must use the venv's python3 path, not system python
